# 最后提交怎么做

现在项目已经准备到“只差真实作者信息”的状态。

## 你只需要改一个文件

文件：

```text
paper/author_metadata_template.json
```

把里面的占位内容换成真实信息：

```json
{
  "authors_latex": "Feiyang Song\\inst{1} \\and Student A\\inst{1} \\and Student B\\inst{1}",
  "authorrunning": "F. Song et al.",
  "institute": "School of ..., ... University, City, China",
  "email": "your_email@example.com",
  "acknowledgements": ""
}
```

## 作者字段怎么写

一个单位：

```text
Feiyang Song\inst{1} \and Student A\inst{1} \and Student B\inst{1}
```

多个单位：

```text
Feiyang Song\inst{1} \and Student A\inst{1} \and Teacher B\inst{2}
```

多个单位时，`institute` 可以写：

```text
School of ..., University A, City, China \and Department of ..., University B, City, China
```

## 改完后运行一条命令

在 `C:\Users\93785\Desktop\CIVS` 目录运行：

```powershell
python scripts\prepare_final_submission.py --metadata paper\author_metadata_template.json
```

成功后会生成：

```text
paper/RaOD-ERAS_CCIS_draft.pdf
dist/raod_eras_ccis_submission_package.zip
dist/raod_eras_release.zip
```

## 如果失败怎么看

如果看到：

```text
author_metadata_replaced failed
```

说明作者、单位、邮箱里还有占位词，比如：

```text
First Author
Second Author
Corresponding Author
Institution Name
author@example.com
```

把这些都替换掉，再运行同一条命令。

## 提交系统里复制什么

从这个文件复制：

```text
paper/submission_form_fields.md
```

里面有：

```text
Title
Short Title
Abstract
Keywords
Dataset Statement
Code Availability
```

从这个文件复制声明：

```text
paper/statements.md
```

里面有：

```text
Data Availability
Ethics Statement
Conflict of Interest
Funding
Acknowledgements
Limitations
```

## 最终上传哪个包

会议系统如果只要 PDF，上传：

```text
paper/RaOD-ERAS_CCIS_draft.pdf
```

会议系统如果要 LaTeX source，上传：

```text
dist/raod_eras_ccis_submission_package.zip
```

GitHub 或代码附件用：

```text
dist/raod_eras_release.zip
```
