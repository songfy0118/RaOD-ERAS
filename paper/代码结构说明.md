# 代码结构说明

现在工程分两层：

1. `scripts/`：命令行入口，负责启动实验和作图。
2. `src/raod_eras/`：正式论文方法代码，负责数据、模型、指标、消融和报告。

## 新增正式模块

| 文件 | 作用 |
|---|---|
| `src/raod_eras/config.py` | 实验配置：数据路径、输出路径、DINO参数、阈值搜索范围 |
| `src/raod_eras/datasets.py` | SMIYC RoadObstacle 数据集读取，匹配 RGB 和 GT |
| `src/raod_eras/priors.py` | 道路先验、自车道先验、近场权重、归一化 |
| `src/raod_eras/baselines.py` | RoadContrast 低级 baseline |
| `src/raod_eras/dino_features.py` | DINOv2 特征提取和道路原型异常热力图 |
| `src/raod_eras/refinement.py` | ERAS 三种消融版本：light / balanced / recall |
| `src/raod_eras/metrics.py` | AP、F1、IoU、Precision、Recall、FPR95 |
| `src/raod_eras/experiment.py` | 主实验流程：跑所有方法、保存热力图、保存二值图、汇总指标 |
| `src/raod_eras/reporting.py` | 论文结果表和可视化网格 |
| `src/raod_eras/io_utils.py` | 保存 CSV、JSON、热力图、mask |
