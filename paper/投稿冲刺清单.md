# 投稿冲刺清单

## 当前可投稿题目

```text
RaOD-ERAS: Road-Prototype Anomaly Heatmap with Ego-Lane Risk-Aware Refinement for Unexpected Road Obstacle Segmentation
```

中文解释：

```text
面向道路突发异物的训练自由热力图生成与自车道风险感知分割方法。
```

## 论文核心贡献

1. 不训练新模型，调用 DINOv2 预训练特征，构建 road-prototype anomaly heatmap。
2. 提出 ERAS，用道路区域、自车道、近场权重和连通域风险评分改进热力图。
3. 输出三类结果：热力图、二值分割图、可反馈给下游系统的 warning event JSON。
4. 在 SMIYC RoadObstacle、RoadAnomaly21、StreetHazards partial 149 张上验证。

## 不能声称

```text
不能说 SOTA。
不能说完整 StreetHazards benchmark。
不能说训练了新的大模型。
不能说对所有数据集全面提升。
```

## 当前实验结论

| Dataset | 样本数 | 最适合写进论文的结论 |
|---|---:|---|
| SMIYC RoadObstacle | 30 | DINO+ERAS light AP/FPR95 最好，是主实验 |
| RoadAnomaly21 | 10 | DINO+ERAS balanced 提升 F1/IoU/FPR95，支持泛化 |
| StreetHazards partial | 149 | raw DINO AP/F1 更好，ERAS light FPR95 更好，只能写困难域误报控制 |
