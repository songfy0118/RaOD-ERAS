# CCIS Submission Notes

Springer states that CCIS uses the same Computer Science proceedings instructions and templates as LNCS/LNAI/LNBI/LNBIP/LNICST/IFIP AICT.

Official template page:

```text
https://link.springer.com/series/558/information-for-authors-and-editors
```

Use the LaTeX2e proceedings template ZIP from that page. Put `paper_ccis_latex.tex`, `references.bib`, and the `figures/` folder into the downloaded template directory. The LaTeX file uses:

```text
\documentclass[runningheads]{llncs}
\bibliographystyle{splncs04}
```
