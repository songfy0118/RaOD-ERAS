# Paper Statements

## Data Availability

```text
This work uses publicly available benchmark data or locally prepared public subsets. Due to dataset size and redistribution restrictions, raw datasets are not included in the code release. The repository documents the expected local directory structure and reproduction commands.
```

## Code Availability

```text
The RaOD-ERAS implementation, experiment scripts, paper source, figures, and reproduction instructions are prepared for release. Large datasets, pretrained model caches, and generated dense outputs are excluded.
```

## Ethics Statement

```text
The work studies road anomaly segmentation for intelligent driving safety. It is intended for research and warning-output analysis rather than direct deployment in safety-critical vehicles. The method may produce false positives or false negatives and should not be used as a standalone driving decision module.
```

## Limitations

```text
The public GT subsets of SMIYC and RoadAnomaly21 are small. StreetHazards was only partially downloaded, and the reported 149-sample result is not an official benchmark number. The method relies on geometric road priors and may fail under unusual camera viewpoints or non-road scenes. It also does not replace supervised anomaly segmentation models.
```
